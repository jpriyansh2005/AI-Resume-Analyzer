# 🚀 ATS Pro — AI Resume Analyzer

ATS Pro is an AI-powered resume analysis platform built for **Computer Science students and developers** to optimize their resumes for **ATS systems, recruiter screening, and technical interviews**.

It analyzes resumes, identifies weak sections, detects missing technical skills, suggests improvements, and provides an interactive recruiter-style report.

---

## ✨ Features

### 📊 ATS Score Analysis
- ATS compatibility scoring out of 100
- Resume readability and recruiter friendliness analysis
- ATS keyword density evaluation

### 🧠 AI Resume Analysis
- AI-generated recruiter feedback
- Resume strengths and weaknesses detection
- Personalized improvement recommendations

### 🔥 Dynamic Resume Heatmap *(Signature Feature)*
- Visual resume section analysis
- Highlights strong, average, and weak areas
- Dynamic detection based on actual resume sections

### 🎯 Skill Gap Detection
- Detects missing technical skills
- Suggests important technologies for target roles
- Priority-based recommendations

### ✍️ Project Description Optimization
- Converts weak project descriptions into recruiter-friendly bullet points
- Improves ATS keyword matching
- Suggests quantified achievements

### 📝 Resume Wording Refinement
- Grammar improvements
- Better technical wording suggestions
- ATS-friendly resume optimization

### 📈 Interview Readiness Insights
- Resume quality scoring
- Skill matching analysis
- Recruiter readiness estimation

### 🕒 Resume History Tracking
- Stores previous resume analyses
- Track profile improvement over time

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/jpriyansh2005/AI-Resume-Analyzer.git
```

### 2. Navigate to Project

```bash
cd AI-Resume-Analyzer
```

### 3. Install Frontend Dependencies

```bash
cd client
npm install
```

### 4. Install Backend Dependencies

```bash
cd ../server
npm install
```

### 5. Create Environment Variables

Create a `.env` file inside the **server** folder:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
PORT=5000
```

### 6. Run Backend

```bash
cd server
npm run dev
```

### 7. Run Frontend

```bash
cd client
npm run dev
```

## 💡 Why I Built This

As a Computer Engineering student, I noticed that many students struggle to understand **why their resumes get rejected**.

Most resume tools only provide a score without meaningful guidance.

I built **ATS Pro** to provide:
- Recruiter-style resume feedback
- AI-powered improvement suggestions
- Resume heatmap visualization
- Skill-gap detection for developer roles

The goal is to help students become **interview-ready** with actionable feedback.
